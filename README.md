# NLP Course Materials and Assignments

这是一个包含自然语言处理（NLP）课程实验和作业的项目仓库。

## 📁 项目结构

```
nlp/
├── NLP-Lab1/                          # 实验一：正则表达式
│   ├── NLP-Lab1-student.ipynb        # 学生实验手册
│   ├── pic/                          # 实验相关图片资料
│   └── test.py                       # 测试代码
├── NLP-Lab2-Regular Expression2.pptx # 实验二：正则表达式进阶
├── NLP-Lab2-Regular Expression2-student.ipynb # 学生实验笔记
├── NLP-Lab3-student/                  # 实验三：文本处理
│   ├── NLP-Lab3-TextProcess-student.ipynb # 文本处理实验
│   ├── README.md                     # 实验说明文档
│   ├── data/                         # 实验数据集
│   ├── dict/                         # 词典文件
│   └── deal_lab3.py                  # 数据处理脚本
├── NLP-Assign-01-stu/                 # 作业一
│   ├── NLP-Assign-01-stu.ipynb       # 作业一题目
│   └── images/                       # 作业相关图片
└── NLP-Assign-02-stu/                 # 作业二：n-gram模型
    ├── n-gram stu.ipynb              # 作业二题目
    └── gaicibi.txt                   # 文本数据
```

## 📚 内容说明

### 实验部分

#### 🧪 实验一：正则表达式基础
- **文件**: `NLP-Lab1/NLP-Lab1-student.ipynb`
- **内容**:
  - 正则表达式基础语法
  - 模式匹配练习
  - 文本提取技巧
  - 配套图片和示例

#### 🔧 实验二：正则表达式进阶
- **文件**: `NLP-Lab2-Regular Expression2-student.ipynb`
- **内容**:
  - 复杂正则表达式模式
  - 实际应用场景
  - 性能优化技巧

#### 📝 实验三：文本处理
- **文件**: `NLP-Lab3-student/`
- **内容**:
  - 文本预处理技术
  - 分词与词性标注
  - 数据集处理流程
  - 包含真实数据文件和处理脚本

### 作业部分

#### 📋 作业一
- **文件**: `NLP-Assign-01-stu/`
- **内容**: 基础NLP概念和工具使用

#### 📊 作业二：n-gram模型
- **文件**: `NLP-Assign-02-stu/`
- **内容**:
  - n-gram语言模型实现
  - 文本数据分析
  - 包含中文文本数据集

## 🛠️ 环境要求

- Python 3.7+
- Jupyter Notebook
- 必要的Python库：
  ```
  pandas
  numpy
  jupyter
  matplotlib
  re
  nltk
  jieba (中文分词)
  ```

## 🚀 使用方法

1. 克隆仓库：
   ```bash
   git clone https://github.com/Zouksw/nlp.git
   cd nlp
   ```

2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 启动Jupyter Notebook：
   ```bash
   jupyter notebook
   ```

4. 按照实验和作业文件中的说明逐步完成练习。
